# 🎮 Pokémon Memory Card Game

A fun, interactive memory-testing game built with React and Vite. The application fetches data from the public PokeAPI to dynamically display collectible card components. The goal is to challenge your memory by clicking each Pokémon card only once!

## ✨ Features
* **Dynamic API Integration:** Fetches 12 unique Pokémon sprites and names seamlessly from the [PokeAPI](https://pokeapi.co/).
* **Real-time Shuffling:** Instantly randomizes card positioning using a custom array-shuffling algorithm every time a card is selected.
* **Score Tracking:** Keeps active record of the player's `Current Score` alongside their all-time `High Score` using managed React state.
* **Responsive Styling:** Crafted with custom vanilla CSS and Grid layouts to ensure fluid gameplay across mobile, tablet, and desktop devices.
* **Component-Driven Architecture:** Clean codebase structured using modular parent-child components (`Header`, `CardGrid`, and `Card`).
